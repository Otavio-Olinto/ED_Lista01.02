# ED_Lista01.02
## Menor valor de um vetor

Este desafio consiste em criar uma aplicação, em Java, que receba um vetor de interios e apresente qual é o menor valor presente neste vetor.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. A vrificação do menor valor deve ser realizada por uma função recursiva;
3. Esta função deve receber como parâmentos o vetor, seu tamanho e o valor de sua última posição;
4. Declarar, via comentário, a condição de parada da função.
```
